C'est vraiment gratuit ? +
Oui. Le plan Gratuit fonctionne avec votre propre clé API Hugging Face (100% gratuite). Scriptly ne vous facture rien pour l'utiliser. Vous payez uniquement si vous voulez les modèles premium et les fonctionnalités avancées du plan Pro.
Comment obtenir une clé Hugging Face ? +
Créez un compte sur huggingface.co (gratuit), allez dans Settings → Access Tokens, créez un token de type "Read". C'est tout. La clé commence par "hf_" et fonctionne immédiatement dans Scriptly.
Le contenu généré est-il détectable comme IA ? +
Scriptly génère du contenu naturel et conversationnel. En choisissant votre ton et en personnalisant les prompts, le résultat est très difficile à distinguer d'une rédaction humaine. Des milliers d'utilisateurs publient chaque jour sans jamais mentionner l'IA.
